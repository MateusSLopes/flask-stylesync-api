# Flask Sales API

Este projeto é uma API REST construída com Flask.

## Estrutura
- `app/`: Contém o código principal da aplicação (modelos, rotas, decoradores).
- `config.py`: Configurações da aplicação.
- `run.py`: Ponto de entrada para executar a aplicação.

## Requisitos
- Python 3.x
- Flask
- Outras dependências listadas na configuração do ambiente.

## Executando a Aplicação
Primeiro, instale as dependências:

```bash
pip install -r requirements.txt
```

Para rodar a API, ative o ambiente virtual e execute:

```bash
python run.py
```
